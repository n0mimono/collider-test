## 各ボタン

* Model/Col: オブジェクト切り替え（Cube（左）, Sphere（右））
* Model/Cas: キャスター切り替え（Ray、Sphere）
* Cast: キャストを飛ばす
* Clear: オブジェクトを全て削除する
* +1: オブジェクトを1個追加
* +10: オブジェクトを10個追加

## 右下テキスト

* Cols: コライダーの数
* Cast: キャスター切り替え状態
* Col: コライダー切り替え状態
* Nums: キャスト回数
* Hit: キャストがヒットしたか否か
* Time: キャスト時間
